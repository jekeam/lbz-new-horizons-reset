# Сброс ЛБЗ «Новые горизонты»

Клиентский UX-мод для «Мир танков» 1.43.0.0, который показывает в ангаре небольшую панель активных ЛБЗ «Новые горизонты» и ускоряет переход к штатному сбросу задачи.

Мод не меняет прогресс напрямую и не обходит серверную логику. Обычный клик открывает карточку задачи, `Alt + клик` вызывает штатный клиентский путь сброса с серверной проверкой.

## Установка

1. Скачай релизный архив `lbz-new-horizons-reset-1.43.zip`.
2. Распакуй его в корень игры, например:

```text
C:\Games\Tanki\
```

После распаковки файл должен лежать так:

```text
C:\Games\Tanki\mods\1.43.0.0\lbz_new_horizons_reset_1.43.mtmod
```

Если в этой папке есть старые версии этого мода, удали их.

## Управление

- `F9` - показать или скрыть панель.
- Клик по задаче - открыть карточку задачи.
- `Alt + клик` по задаче - запустить штатный сброс.
- `Ctrl + ЛКМ` по панели - перетащить панель.

Панель показывается только в ангаре. В бою, очереди, пребое, на логине и при смене сервера она скрывается.

## Сборка

Требуется Python 2.7 x64 клиента. По умолчанию скрипт ждёт его здесь:

```text
C:\Python27\python.exe
```

Собрать `.mtmod` и релизный архив:

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\build_release.ps1
```

Результаты:

```text
_build\release_1.43\lbz_new_horizons_reset_1.43.mtmod
release\1,43\lbz-new-horizons-reset-1.43.zip
```

Архив уже содержит нужную структуру:

```text
mods\1.43.0.0\lbz_new_horizons_reset_1.43.mtmod
```

## Логи для разработчиков

По умолчанию логирование минимальное: только ошибки через `print`, они попадают в:

```text
C:\Games\Tanki\python.log
```

Отдельный лог-файл мод не создаёт.

Чтобы временно расширить логи для разработки, в `mod_lbz_dynamic_reset.py` поставь:

```python
DEBUG_LOG = True
```

После этого пересобери мод скриптом выше. Debug-сообщения также будут идти только в `python.log`.
